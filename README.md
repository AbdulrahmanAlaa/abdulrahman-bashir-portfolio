## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/AbdulrahmanAlaa/abdulrahman-bashir-portfolio.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm start
```

4. Build for production:

```bash
npm run build
```
